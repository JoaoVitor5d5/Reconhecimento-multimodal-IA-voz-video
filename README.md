Este projeto necessita de algumas ferramentas, o mesmo foi configurado com o vscode, e para isso, se utilizou:
- python 3.13.5
- pip 25.1.1
- cmake (https://cmake.org/)
- visual studio c++ (https://visualstudio.microsoft.com/pt-br/vs/features/cplusplus/)

Para rodar, configure usando:
- pip install -r requirements.txt


Após, para configurar seu rosto e voz, use o comando:
- Python register.py

Irá pedir para que se posicione na frente da webcam, e depois que fale algo por 5 segundos (indico que diga seu nome completo)

E para testar, rode:
- Python main.py

Novamente irá pedir para se posicionar na frente da webcam e falar algo por 5 segundos

(Observação: É um projeto inicial, então ainda possui erros de reconhecimento, mas funciona em pequenos ambientes silenciosos)
